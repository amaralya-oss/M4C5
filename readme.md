# Actividad M4C5

## Descripcion del proyecto

Este programa **permite la inscripcion de jugadores de Bascketball para un torneo**

El usuario debera ingresar los siguientes datos

-Nombre
-Edad
-Altura
-Posicion preferida

Luego el programa **verifica si cumple los requisitos para la inscripcion** y registra los datos

## Requisitos

-Edad menor a 15 años 
-Altura minima de 1.60
-Posicion existente
-Todos los campos completos
**clasifica a categoria juvenil**

-Edad mayor a 15 años
-Altura superior a 1.60
-Posicion existente
-Todos los campos completos
**clasifica a categoria adulto**

El usuario recibe un **alert* con informacion al presionar el boton registrar
